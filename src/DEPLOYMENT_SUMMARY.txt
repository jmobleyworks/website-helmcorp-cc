================================================================================
ARCHITECTURE HIGHLIGHTS
================================================================================

Pure Functional Design:
- All handlers use Either/Option monads for type-safe error handling
- Composition via pipe() and asyncPipe() for middleware stacking
- No side effects outside fetch handler
- Testable functors (no implicit state)

Route Pattern Matching:
- Dynamic route matching with parameter extraction
- /api/config/:domain → getConfig(domain)
- /api/history/:domain → getHistory(domain)
- Extensible for future routes

Middleware Pipeline:
  Request
    ↓
  [Validation] ← Either monad, returns 400 if invalid
    ↓
  [Logging] ← tap() side effect, logs to console
    ↓
  [Routing] ← routeRequest dispatcher
    ↓
  [Handler] ← Either.fold for error/success paths
    ↓
  [Timing] ← addTimingHeader adds X-Response-Time-Ms
    ↓
  [Cache] ← applyCacheHeaders based on pathname
    ↓
  [CORS] ← addCORS with Access-Control headers
    ↓
  Response

Error Handling:
- Request validation: validateRequest() → Either
- Environment validation: validateEnvironment() → Either
- Database queries: queryDB() → Either
- Route matching: matchPattern() → params|null
- Fold pattern: result.fold(onError, onSuccess)

Caching Strategy:
- Home/Config routes: max-age=300, stale-while-revalidate=3600
- API routes: max-age=60, stale-while-revalidate=300
- Health/Stats: no-cache, max-age=0

================================================================================
KEY FUNCTORS REFERENCE
================================================================================

Either Monad:
  Either.Right(value)        → Success container
  Either.Left(error)         → Error container
  result.fold(onError, onSuccess)  → Pattern match

Option Monad:
  Option.Some(value)         → Value present
  Option.None                → Value missing
  result.getOrElse(default)  → Unwrap with default

Composition:
  compose(f, g, h)(x)        → f(g(h(x))) [right-to-left]
  pipe(f, g, h)(x)           → h(g(f(x))) [left-to-right]
  asyncPipe(f, g, h)(x)      → await each result

Higher Order:
  curry(fn)                  → Function(arg1)(arg2)...
  partial(fn, arg1)          → Function(arg2, arg3)...
  memoize(fn)                → Cache results by arguments
  retry(maxTries, delay, fn) → Exponential backoff

Utilities:
  tap(fn)(value)             → Execute fn, return value
  once(fn)                   → Execute only first call
  debounce(ms, fn)           → Delay execution
  throttle(ms, fn)           → Limit frequency
